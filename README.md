# 📊 Power BI Sales Dashboard

## 📌 Project Overview

This project presents an interactive sales dashboard built using Power BI to analyze sales performance across products and regions.

## 🛠️ Tools Used

* Power BI
* Microsoft Excel

## 📈 Key Features

* KPI Card displaying Total Sales
* Bar Chart for Sales by Product
* Pie Chart for Sales Distribution
* Interactive Filters (Slicers) for Region and Product

## 🔍 Insights

* Identified top-performing products
* Compared sales across regions
* Enabled dynamic filtering for better analysis

## 📁 Files Included

* `Sales_Dashboard_PowerBI.pbix` → Power BI dashboard file
* Dashboard screenshot (optional)

## 🚀 Outcome

Built a professional and interactive dashboard to visualize sales data and derive meaningful insights.
